## Java Course Tasks

### **Session 1**

**Hello World**

- Create a simple non-GUI application that prints the following text on the command prompt:  
    `Hello Java`
    

**Even or Odd**

- Create a simple program to determine whether a given number is even or odd.
    

**IP Separation**

- Create a non-GUI application that accepts a well-formed IP address as a string and separates it into parts using the dot (`.`) delimiter.
    

---

### **Session 2**

**Student Info**

- Create a non-GUI application that represents a `Student` and includes a method to print student information.
    

**Complex Number**

- Create a non-GUI application that represents a complex number.
    
- Include methods to:
    
    - Add complex numbers
        
    - Subtract complex numbers
        
    - Print complex numbers
        

---

### **Session 3**

**Shape Class**

- Create an abstract `Shape` class with:
    
    - A `dim` attribute
        
    - An abstract method `calculateArea()`
        
- Create three subclasses:
    
    - `Rectangle`
        
    - `Triangle`
        
    - `Circle`
        
- Create a `Picture` class with a method `sumAreas()` that accepts any three shapes and returns the sum of their areas.
    

**Override Example**

- Create a `Parent` class with:
    
    - Two attributes: `num1` and `num2`
        
    - A method `sum()` that calculates their total
        
- Create a `Child` class that:
    
    - Inherits from `Parent`
        
    - Has an additional attribute `num3`
        
    - Overrides the `sum()` method to calculate the total of all three numbers
        

---

### **Session 4**

**Lamp Draw**

- Create a `JFrame` application that uses the `Graphics` class to draw a lamp shape.
    

---

### **Session 5**

**Date and Time JFrame**

- Create a `JFrame` that displays the current date and time.
    

**Text Banner JFrame**

- Create a `JFrame` that displays a marquee (scrolling) text string.
    

---

### **Session 6**

**Button Count**

- Create a `JFrame` with two buttons:
    
    - One to increment a counter value
        
    - One to decrement the counter value
        

**Draw Single Line**

- Create a `JFrame` that allows the user to draw a single line by dragging the mouse.
    

---

### **Session 7**

**TextPad**

- Create a text editor application.
    
- Implement open and save functionality using **one** of the following:
    
    - `FileInputStream` and `FileOutputStream`
        
    - `DataInputStream` and `DataOutputStream`
        
    - `FileReader` and `FileWriter` _(Bonus)_
        

---

### **Final Project**

**Paint Brush**

- Create a `JFrame`-based paint application that allows the user to:
    
    - Draw basic shapes
        
    - Choose different colors
        
    - Clear all drawings
        
    - Erase specific parts of drawings
