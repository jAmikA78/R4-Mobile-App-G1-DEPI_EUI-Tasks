##### [Shape clas](./Java-Tasks/Session-03/shapes/)

Shape class which have a dim attribute and an abstract calculateArea( ), the shape will have three subclasses Rectangle, Triangle and Circle. Finally, create a Picture class which will have one method sumAreas( ) which accepts any three shapes as parameters and return the total sum of their areas.

##### [Override Example](./Java-Tasks/Session-03/overrideExample/)
